Question-1: What are the different ways to select an element in the DOM?

We can select elements in several ways, such as getElementById, getElementsByClassName, querySelector, querySelectorAll, and getElementsByTagName.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML: It gives html code which inside the tag.
2. innerText: It gives only the text which inside the tag.
3. textContent: It gives all the text inside the tag, including hidden elements.

Question-3: What is event delegation in the DOM?

Instead of adding event listeners to multiple child elements,add one listener to the parent and parent element handles events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event triggered on a child element propagates up through its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?
